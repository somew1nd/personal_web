Powerful transfer learning method: Network Global Encoding

May 2023 - Present
     
Southwest Minzu University
     
Assistant researcher

ChengDu, China

Currently, in engineering, migration learning is a vital tool for the practical application of machine learning. The current migration learning methods fail to utilize efficiently the structural characteristics of the source network. We utilize word embedding in natural language processing to encode neural network structures. In migration learning, our goal is to maintain the Euclidean distance between the original and target networks for improved assistance.

My tasks involves:
First, explore the similarity of output features between the source and target networks, and then measure the similarity of parameter distribution features of the two networks. Second, incorporate the Euclidean distance of the embedding of the two networks into the reconstruction of the loss function. Third, construct the network embedding generation structure, and improve the Transformer. Finally, conduct experiments and write the paper.
